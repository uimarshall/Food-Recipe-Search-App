# Quick start

## clone repository

`git clone https://github.com/uimarshall/Food-Recipe-Search-App`

##change directory

`cd Food-Recipe-Search-App`

## Install dependencies for server

npm install

## Run the React app

npm start

## Server runs on http://localhost:3000
